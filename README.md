# 📊 Smart Student Dashboard

The **Smart Student Dashboard** is a modern and responsive productivity web application designed for students to manage tasks, save notes, organize study essentials, and improve productivity — all in one place.

This project is built using **HTML, CSS, and JavaScript** with **LocalStorage** support for saving data permanently in the browser.

---

# 🚀 Features

## ✅ Task Management System

- Add new tasks
- Add optional due dates
- Mark tasks as completed
- Delete tasks
- Filter tasks:
  - All Tasks
  - Pending Tasks
  - Completed Tasks
- Data stored using LocalStorage

---

## 📝 Notes Section

- Save quick study notes
- Notes remain saved after page refresh
- Uses browser LocalStorage

---
